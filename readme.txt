== Description ==

Add a vimeo player with a list of all videos in the album that load 
into player on click.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `vimeo-album` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. add shortcode tag [vimeoAlbum album_id="your_album_id"]
